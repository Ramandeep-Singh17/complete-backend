# 🚀 Practice Full Stack App

A full-stack web application built using modern technologies with a clean frontend and a functional backend. This project demonstrates end-to-end development including API creation, frontend integration, and data handling.

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/Ramandeep-Singh17/PracticeProjectFullstackApp.git
cd PracticeProjectFullstackApp
```

---

### 2️⃣ Setup Backend

```
cd backend
npm install
npm start
```

---

### 3️⃣ Setup Frontend

```
cd frontend
npm install
npm run dev
```
