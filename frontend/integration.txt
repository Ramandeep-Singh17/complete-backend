# FULLSTACK INTEGRATION NOTES (Frontend + Backend)

---

1. BASIC FLOW (HIGH LEVEL)

---

Frontend (UI)  --->  Backend (API)  --->  Database

User action → Frontend request → Backend process → DB → Response → UI update

---

2. STEP BY STEP PROCESS

---

STEP 1: Backend API banana

Backend me hum routes banate hain:

Example:
GET /posts       → data fetch
POST /posts      → data create

Example (Node.js):
app.get("/posts", (req, res) => {
res.json({ posts: [...] })
})

---

STEP 2: Frontend setup

Frontend me:

* axios install karte hain

Command:
npm install axios

---

STEP 3: API call karna (Frontend → Backend)

Example:

import axios from "axios"

useEffect(() => {
axios.get("http://localhost:3000/posts")
.then((res) => {
console.log(res.data)
})
}, [])

---

STEP 4: Data ko state me store karna

const [posts, setPosts] = useState([])

axios.get(...)
.then((res) => {
setPosts(res.data.posts)
})

---

STEP 5: UI me render karna

{posts.map((post) => ( <div key={post._id}> <img src={post.image} /> <p>{post.caption}</p> </div>
))}

---

3. POST REQUEST (DATA SEND KARNA)

---

Frontend:

//Frontend se hum backend ki API call karte hain, jo humne backend me banayi hoti hai 
//Frontend → request bhejta hai → Backend API → response deta hai 

axios.post("http://localhost:3000/posts", {
caption: "Hello",
image: "url"
})

Backend:

app.post("/posts", (req, res) => {
const { caption, image } = req.body
// save to DB
res.json({ message: "Post created" })
})

---

4. IMPORTANT CONCEPTS

---

1. API = bridge between frontend & backend

2. Frontend:

   * UI dikhata hai
   * request bhejta hai

3. Backend:

   * logic handle karta hai
   * DB se data fetch/save karta hai

4. Axios:

   * HTTP client hai
   * GET, POST, PUT, DELETE request bhejta hai

---

5. COMMON MISTAKES

---

❌ Wrong URL
❌ res.data vs res.data.posts confusion
❌ CORS issue
❌ backend server run nahi ho raha
❌ image URL wrong

---

6. REAL LIFE FLOW (EXAMPLE)

---

User click "Create Post"

→ Frontend axios.post()
→ Backend data receive karta
→ DB me save hota
→ Response return
→ Frontend UI update

---

7. FINAL SUMMARY

---

Backend:

* API banata hai

Frontend:

* API call karta hai

Axios:

* dono ko connect karta hai
